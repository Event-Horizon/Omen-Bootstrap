#Omen Bootstrap#

##Description##
This is my personal Bootstrap project. 
I intend to create a simpler form of the Twitter Bootstrap. 
**Planned:**
 - Buttons
 - Platforms
 - Menus
 - Columns.
I may add more in the future.

Included resources are JQUERY.JS (1.7.2) and BOX-SHADOW.JS.
